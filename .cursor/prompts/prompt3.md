Добавь в существующее React-приложение (3D калькулятор кастомизации мерча) возможность загружать свою 3D-модель.

## Где разместить:
Под кнопкой "Загрузить принт" добавить кнопку **"Загрузить 3D модель"**.

## Поведение:
1. Пользователь нажимает кнопку, выбирает файл (форматы: .glb, .gltf, .obj).
2. Выбранная модель **заменяет** текущее встроенное изделие (зонт/кепка/футболка/кружка/пакет) в центральной 3D-сцене.
3. Модель отображается в том же месте, с теми же возможностями:
   - вращение (OrbitControls)
   - наложение принта (выбранное животное или загруженный пользователем принт)
   - изменение цвета модели (если у модели есть материалы, которые можно перекрасить)
4. Если у загруженной модели нет подходящих материалов для перекраски — цвет не применяется или применяется только к основному материалу.

## Дополнительно:
- Добавить кнопку **"Сбросить"** рядом с "Загрузить 3D модель", которая возвращает встроенную модель (последний выбранный тип изделия из левого меню).
- Пока модель загружается — показывать спиннер/лоадер.
- Если файл некорректный — показывать ошибку и не менять текущую модель.

## Взаимодействие с существующей логикой:
- Левое меню (выбор типа изделия) должно работать только когда активна встроенная модель. Если загружена пользовательская модель — левое меню временно отключается или игнорируется.
- При нажатии "Сбросить" — восстанавливается активный тип изделия из левого меню.

## Технически:
- Использовать `useLoader` из `@react-three/fiber` или `useGLTF` из `@react-three/drei` для загрузки.
- Для .obj — использовать `OBJLoader`.
- Управлять состоянием: `isCustomModel` (true/false) и `customModelUrl` или сам объект модели.

Код должен быть конкретным, с обработкой ошибок и загрузки.
